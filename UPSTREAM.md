# Upstream Provenance

64DOS uses PDOS as the public-domain reference base for the project direction
and DOS-style interface goals.

Pinned upstream:

- Project: PDOS - Public Domain Operating System
- Homepage: https://www.pdos.org/
- Source tree: https://sourceforge.net/p/pdos/gitcode/ci/master/tree/
- Git URL: https://git.code.sf.net/p/pdos/gitcode
- Commit: `c5ebdcfa75aeb3819ca903bb00f624f0ec83b01d`

The PDOS tree describes itself as released to the public domain / CC0. The
selected provenance note in `upstream/pdos/README.pinned.txt` records the
exact commit and the public-domain statement used for this implementation.

64DOS v1 does not copy FreeDOS, GPL, or Microsoft MS-DOS code. The bootloader,
kernel, FAT12 image builder, and tests in this repository are new
public-domain implementation work.
