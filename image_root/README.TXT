64DOS boots through BIOS, enters x86-64 long mode, and runs a tiny
DOS-style shell from a 1.44 MB FAT12 floppy image.

This v1 image is read-only after boot. Try HELP, DIR, LS, TYPE
README.TXT, CAT README.TXT, DUMP README.TXT 64, RUN DEMO.BAT,
DATE, TIME, INFO, COLOR 1E, PROMPT 64DOS>, CLS, MEM, ECHO text,
or REBOOT.
